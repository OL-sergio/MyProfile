# MyProfile

MyProfile is a simple personal website built with **HTML**, **CSS**, and **JavaScript**, styled primarily using the **Bootstrap** framework. It is designed to be responsive, clean, and easy to maintain.

## Tech Stack

- **HTML5** – Structure of the pages.
- **CSS3** – Custom styles and layout tweaks.
- **Bootstrap** – Main UI framework for layout and components.

## Bootstrap Framework

This project uses [Bootstrap](https://getbootstrap.com/) to:

- Provide a responsive grid system (`container`, `row`, `col-*` classes).
- Style typography, buttons, forms, and navigation.
- Use ready-made components like:
  - **Navbar** for the header and navigation.
  - **Cards** for sections like profile, projects, or skills.
  - **Buttons** for links and calls to action.
  - **Forms** for contact or feedback sections.
- Add utility classes for spacing, colors, and alignment.

## Main Components (Pages/Sections)

Typical sections of the website include:

- **Header / Navbar**
  - Built with Bootstrap `navbar` component.
  - Contains the site logo or name and navigation links to each section.

- **Hero / Introduction**
  - Short introduction about you.
  - Often includes a headline, a short description, and a call-to-action button.

- **About**
  - Information about your background, interests, and profile.
  - May use Bootstrap grid and cards to organize content.

- **Projects / Portfolio**
  - List of selected projects.
  - Implemented using Bootstrap **cards** in a responsive grid.

- **Skills**
  - Highlights technical skills (programming languages, frameworks, tools).
  - May be shown with progress bars, badges, or simple lists.

- **Contact**
  - Contact form (name, email, message) or links to social networks.
  - Uses Bootstrap form components and validation styles.

- **Footer**
  - Copyright, social links, or a simple closing line.
  - Styled with Bootstrap utilities for background, text color, and spacing.

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/OL-sergio/MyProfile.git
   cd MyProfile
   ```

2. **Open the website:**

   Open `index.html` in your browser:

   - Double-click `index.html`, or
   - Serve it locally (for example, with VS Code Live Server extension).

## Customization

- Update text and images directly in the HTML files (e.g., `index.html`).
- Adjust colors, fonts, and spacing in the main CSS file (for example, `style.css`).
- Add or remove Bootstrap components by editing the HTML structure and classes.
